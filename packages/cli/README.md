# haste-cli

## Overview

The cli tool can be used by Haste Arcade developers to analyze logs, interact with the haste platform, and increase velocity of development of an ILP game.

See [here](https://github.com/playhaste/haste-sdk/blob/main/README.md) for an overview of the haste-sdk repository.
